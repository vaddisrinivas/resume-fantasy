# resume-fantasy

Goals 
- Create a modern-looking modular reactjs/nextjs based resume/portfolio template compatible with [JSON Resume](https://jsonresume.org/)
- Should be able to template cards and card actions
- Must have various card templates to pick from
- Adaptable and future proof
- Easy to generate form builder
- Easy CLI based tool for page creation
- CLI and UI based resume export feature
- 1 click deployment and easy maintenance
- Easier to develop and support.
- Follows freemium
